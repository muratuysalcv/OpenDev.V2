Advanced Search

When advanced search is on, it will
  • allow to search on all/multiple fields
  • show a search drop down button to get to the advanced search
  • allow to use differnet types of search fields
  • allow to choose search operator
  • allow to do All (AND) or Any (OR) logic
  • show search bar when search is applied (2.0+)
  • save searches into local sotrage (2.0+)
